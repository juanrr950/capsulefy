CÁPSULA DEL TIEMPO ONLINE:
PRINCIPAL USO: GESTIONAR TU LEGADO DIGITAL
TAMBIÉN SE PUEDE USAR COMO UNA CÁSPSULA DEL TIEMPO NORMAL Y CORRIENTE.
USO COMO CÁPSULA DEL TIEMPO NORMAL: CREAS UNA "CÁPSULA" EN NUESTRA PÁGINA CON TEXTO, IMÁGENES, VÍDEOS(?)
QUE SERÁN VISIBLES AL PÚBLICO A PARTIR DE UNA FECHA DETERMINADA
USO PARA GESTIONAR EL LEGADO DIGITAL: SE IMPLEMENTA UN INTERRUPTOR DEL HOMBRE MUERTO
SI CADA X TIEMPO(PERSONALIZABLE) LA PERSONA NO DA SEÑALES DE VIDA, LA CÁPSULA SE HACE PÚBLICA
ESTO PUEDE SERVIR PARA 2 COSAS:
DEJAR UN LEGADO DIGITAL CON INFORMACIÓN TUYA, FOTOS, ETC... PARA QUEDAR INMORTALIZADO EN LA RED
DEJAR INFORMACIÓN RELEVANTE PARA TUS SERES CERCANOS. ESTO SE PUEDE APLICAR SOBRETODO A CREDENCIALES EN REDES
SOCIALES Y QUÉ DESEAS QUE SE HAGA CON ELLAS
ADEMÁS, LA PÁGINA SE PUEDE CONECTAR CON OTRAS REDES SOCIALES Y DEJAR UN MENSAJE PERSONALIZADO
SE PODRÍA MANDAR TAMBIÉN CORREOS AUTOMÁTICOS CON LA APERTURA DE DICHA CÁPSULA
USO GRATUITO Y USO DE PAGO:
GRATUITO: LA CÁPSULA EN LA WEB SIMPLEMENTE
DE PAGO: LA CÁPSULA SE CONECTA CON OTRAS APLICACIONES
OTRA OPCIÓN: PERMITIR QUE EL INTERRUPTOR DEL HOMBRE MUERTO SEA ALGO SÓLO DISPONIBLE SI SE PAGA
